# DBZ: Ultimate Duel (PC)

A raw, fast-paced 2D Dragon Ball Z combat engine built exclusively for PC. No matchmaking, no hand-holding—just pure local multiplayer chaos. Grab a keyboard, split the controls, and crush your opponent.

## Key Features
*   **True Local Versus:** Designed strictly for 2-player competitive local matches.
*   **Instant Permadeath Restart:** When you lose, there is no menu. Hit `F5` to instantly reload the battlefield and reset the fight.
*   **Optimized for PC:** Lightweight and engineered for zero input lag on desktop systems.

## Control Mapping

### Player 1 (Left Side)
*   **Movement:** `W` (Up), `A` (Left), `S` (Down), `D` (Right)
*   **Combat (Melee / Kamehameha):** `Z` and `X`

### Player 2 (Right Side)
*   **Movement:** `Arrow Keys` (↑, ←, ↓, →)
*   **Combat (Melee / Kamehameha):** `N` and `M`

## Quick Start
1. Run the executable on Windows/Linux (PC Only).
2. Defeat your opponent.
3. Press `F5` to instantly restart after a knockout.
